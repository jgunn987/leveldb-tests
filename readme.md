key space layout

system

type     | template                             | notes
---------|--------------------------------------|--------------------
metadata | #metadata => {tables, topology, etc} | used to boot the db
log      | #{logId} => {operations}             | replicated log

boot order
* load metadata
* load schemas
* load counters
* serve
